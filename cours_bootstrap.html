class="container mt-5"

container : Classe Bootstrap qui définit un conteneur centralisé avec des marges latérales automatiques pour assurer un bon alignement du contenu.

mt-5 : Classe Bootstrap qui ajoute une marge supérieure (margin-top) égale à 5 unités (une unité Bootstrap est généralement de 0.25rem, soit 20px pour mt-5).

class="col-md-4 mx-auto

col-md-4 Définit une colonne qui occupe 4/12 de la largeur sur les écrans de taille moyenne (md, soit ≥768px). Sur les écrans plus petits, elle prendra toute la largeur.

mx-auto (margin-left: auto; margin-right: auto;), ce qui centre la colonne dans son parent.

class="card shadow"

card Classe Bootstrap utilisée pour styliser une carte avec un design par défaut (bord arrondi, fond blanc).

shadow :Ajoute une ombre à l'élément pour lui donner un effet de relief.

class="card-img-top"

card-img-top Classe Bootstrap qui applique un style spécifique aux images placées au-dessus d'une carte (card). Elle leur donne une largeur de 100% et ajuste leur affichage pour qu'elles restent propres dans la carte.

card-body text-center

card-bodyApplique des espacements intérieurs (padding) à l'intérieur de la carte pour organiser son contenu.

text-centerCentre horizontalement tout le texte contenu dans cette div.

fw-boldRend le texte gras (font-weight: bold;).
